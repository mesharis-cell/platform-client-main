Step 4 captures the single person the logistics team will contact about this specific order — deliveries, timing, any mid-event updates. If that's you, the fields are already filled in. If it's someone else, overwrite them.

- **Contact Person Name** — defaults to your logged-in name, editable.
- **Email Address** — defaults to your login email. Real-time
  validation runs as you type; if the format is invalid, a red
  **Please enter a valid email address** hint shows under the field.
- **Phone Number** — uses an international phone input with a country
  selector. Defaults to UAE (+971). The Continue button stays
  disabled until the number is a valid format for the chosen country.

## Special instructions

Below the contact fields, a **Special Instructions** textarea lets you
leave notes for the crew — branding requirements, venue peculiarities,
VIPs on-site, anything that changes how the install should go. It's
optional.

<Callout kind="tip" title="Think about what the driver and installer will ask">
    "When does the venue open?" "Which entrance?" "Who signs for the drop?" Getting ahead of
    questions here saves phone calls later.
</Callout>
